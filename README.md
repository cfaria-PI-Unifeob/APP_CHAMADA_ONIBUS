# 🚌 APP CHAMADA ÔNIBUS

Sistema desenvolvido para gerenciamento de chamadas e controle de presença de alunos no transporte escolar.

---

# 📚 Projeto Integrado - Desenvolvimento de Sistemas Móveis

Projeto acadêmico desenvolvido utilizando Flutter Web, Node.js, PostgreSQL e integração em nuvem.

---

# 🚀 Tecnologias Utilizadas

## Frontend
- Flutter Web
- Dart

## Backend
- Node.js
- Express.js

## Banco de Dados
- PostgreSQL
- Neon.tech

## Hospedagem
- Render

## Controle de Versão
- Git
- GitHub

---

# 🎯 Funcionalidades

- ✅ Login de usuários
- ✅ Cadastro de alunos
- ✅ Controle de presença
- ✅ Registro de chamadas
- ✅ Integração com banco de dados online
- ✅ Sistema responsivo
- ✅ API REST
- ✅ Armazenamento em nuvem

# ⚙️ Como Executar o Projeto
🔹 Frontend Flutter
```bash
flutter pub get
flutter run -d chrome 
```
🔹 Backend Node.js
 
```bash
npm install
node server.js 
```

# 🌐 Deploy Online
## Frontend

Hospedado no Render utilizando Flutter Web.

## Backend

API REST hospedada no Render conectada ao PostgreSQL do Neon.

## 🗄️ Banco de Dados

O sistema utiliza PostgreSQL hospedado em nuvem através da plataforma Neon.tech.
